# Family Game Rules (家族ゲームルール管理)

家族や友人とトランプゲーム（ポーカー、大富豪など）を遊ぶ際に、その時々の「ハウスルール」を簡単に設定・共有・確認するためのWebアプリです。

## 特徴

- **動的なルール設定**: 
    - 各ゲームのルールを細かくオンオフしたり、数値を設定したりできます。
    - 設定はブラウザ（localStorage）に保存され、次回起動時も維持されます。
- **インテリジェントな表示制御**:
    - 設定の依存関係（例：ジョーカーが「なし」ならジョーカー関連の設定を隠す）を自動で判別し、必要な項目だけを表示します。
- **リアルタイム「遊び方」ガイド**:
    - 選択したルール（プレイスタイルや特殊ルール）に合わせて、ゲームの流れや手順の説明が動的に書き換わります。
- **モバイル最適化**:
    - ゲームを遊びながら手元でサッと確認できるよう、スマホでの閲覧に最適化されたプレミアムなデザインを採用しています。
- **オフラインファースト**:
    - シンプルなHTML/JS/CSS構成で、特別なサーバーなしでも動作します。

## 対応ゲーム

### 🃏 大富豪
- **階級・税金設定**: 順位による呼び名の確認や、カードの引き渡し枚数のカスタマイズ。
- **多彩な特殊ルール**: 8切り、11バック、革命、階段、縛り（数・マーク・激しば）など。
- **反則あがり制限**: ジョーカー、2、8、スペ3あがりの個別禁止設定。

### ♠️ ポーカー
- **プレイスタイル切替**: テキサスホールデムとドローポーカーの切り替えに対応。
- **ベット設定**: ブラインド、アンティ、ベースベット額の細かな設定と、遊び方への自動反映。
- **役の強さ確認**: 標準的な役の強さと、引き分け時の判定ルール。

## 使い方

1. **ルールを設定する**: 各ゲームの「ルール設定」から、遊びたいルールを選びます。
2. **現在のルールを確認する**: 「現在のルールと遊び方」を開くと、設定した内容に基づいた手順が表示されます。
3. **詳細を見る**: 各ルールの「詳細」ボタンから、より詳しい解説（初心者向けガイド）を確認できます。

## 開発者向け情報

### ディレクトリ構造
- `/data`: ゲームごとのルール定義やカテゴリ、依存関係を管理するJS/JSONファイル。
- `/games`: 各ゲームのメイン画面（カテゴリ一覧、設定、現行ルール表示）。
- `/details`: 各ルールの詳細な解説HTML。
- `/scratch`: 開発・テスト用のスクリプト。
- `app.js`: ルール管理、動的表示、データ読み込みのコアロジック。
- `style.css`: 共通のデザインシステムとレスポンシブスタイル。

### ルールの追加方法
`data/` ディレクトリ内の各JSファイルを編集することで、新しいルール項目やカテゴリを簡単に追加できます。表示の依存関係は `data/dependencies.js` で一括管理されています。
