# 配布用ファイルの生成手順（ビルドマニュアル）

スマートフォンで確実に正常表示（オフライン動作）させるため、
LINEなどで家族・友人に配る際は、このプロジェクト内にある複数のファイルを「1つのHTMLファイル」にまとめる必要があります。

この作業は、同梱されている自動化スクリプト（`build.js`）を実行するだけで完了します。

## 準備するもの
- **Node.js** がPCにインストールされている必要があります。
  （まだインストールされていない場合は、[公式サイト](https://nodejs.org/) からダウンロード・インストールしてください）

## 生成手順

1. コマンドプロンプト（または PowerShell、VSCodeのターミナル）を開きます。
2. この `FamilyRules` フォルダ（`build.js` があるディレクトリ）に移動します。
3. 以下のコマンドを入力して実行します。

```bash
node build.js
```

4. `Build successful: dist/FamilyRules_Distribution.html` と表示されれば成功です。
5. 新しく作られた **`dist`** フォルダの中に、**`FamilyRules_Distribution.html`** というファイルが生成されています。

## 配布について
この `FamilyRules_Distribution.html` は、すべてのデータ（画像、デザイン、ルール設定）が内包された「完全自立型の1ファイル」です。

ZIPでまとめる必要はありません。この **1つのファイル（.html）だけをLINE等でそのまま相手に送信**してください。
相手がスマホでファイルをタップすると、どの環境でもデザインが崩れることなく、完全にオフラインでサクサク動作するルールブックが表示されます。

## 今後のメンテナンスについて
- ルールの追加や設定の変更などは、これまで通り `data/` フォルダ内のファイルや各HTMLファイルを編集してください。（開発環境は今まで通りです）
- 修正が終わった後、最後に再度 `node build.js` を実行すれば、最新版の配布用HTMLが生成されます。
